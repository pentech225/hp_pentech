## 4. 画面遷移図

```
[トップ #home]
    │
    ├──→ [プロローグ #prologue]
    │        └──→ P-1 → P-2 → P-3 → P-4 → [第1章へ]
    │
    ├──→ [第1章 #chapter1]
    │        └──→ 1-1 → 1-2 → ... → 1-8 → [第2章へ]
    │
    ├──→ [第2章 #chapter2]
    │        └──→ ゲーム1（STEP1→STEP2→STEP3）→ ゲーム2 → ... → ゲーム5
    │
    ├──→ [第3章 #chapter3] ... ゲーム6〜10
    ├──→ [第4章 #chapter4] ... ゲーム11〜20
    ├──→ [第5章 #chapter5] ... ゲーム21〜30
    ├──→ [付録 #appendix]
    └──→ [タイピング #typing]

※ サイドバーからは任意の章・レッスンに直接ジャンプ可能
※ scratch.mit.edu は常に新タブで開く（このサイトからは離れない）
```

## 6. コンポーネント一覧

| コンポーネント | 用途 | 再利用 |
|--------------|------|--------|
