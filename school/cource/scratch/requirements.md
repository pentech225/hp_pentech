# Scratch学習教材 要件定義書

**プロジェクト名**: スクラッチまなびば  
**作成日**: 2026-04-13  
**ステータス**: 要件定義完了 → 技術要件定義へ

---

## 1. プロジェクト概要

幼稚園児（年中・年長）がスクラッチプログラミングを自習できる教材サイト。  
「スクラッチプログラミング事例大全集」をベースに、HTMLで独自再現する。  
ここさえ見れば、スクラッチが使えるようになることを目標とする。

---

## 2. ユーザー定義

| ロール | 人数 | 属性 |
|--------|------|------|
| 生徒 | 14名 | 年長4名（女性4）、年中10名（男性4・女性6） |
| 先生 | 1名 | PC初心者 |

- 操作: マウス＋キーボード（タッチ操作不要）
- 環境: オンライン（レンタルサーバーで公開）
- デザイン: かわいい系（パステルカラー・キャラクター）

## 3. 機能要件（フェーズ1 — 現在実装）

### 3.1 カリキュラム表示
- 章・レッスン一覧をトップページに表示
- 各レッスンへ遷移できる（同一ページ内のナビゲーション）

### 3.2 ゲームレッスンページ（第2〜5章）— 3ステップ構成

各ゲームレッスンは以下の3ステップで構成する。

```
STEP 1: まず あそんでみよう！
  └ 完成版ゲームをiframeで埋め込み（そのまま遊べる）
  └ 「つぎへ」ボタン

STEP 2: どうやって つくるの？
  └ 作り方の説明テキスト（全漢字ルビ付き）
  └ スクリーンショット画像
  └ ブロックの説明（ビジュアル）
  └ 「つぎへ」ボタン

STEP 3: じぶんで つくってみよう！
  └ 「スクラッチをひらく」ボタン
      → scratch.mit.edu の完成版プロジェクトを新タブで開く
  └ リミックス手順の説明
      ① サインインする（プロローグで作ったアカウント）
      ② 「なかをみる」ボタンをクリック
      ③ 「リミックス」ボタンをクリック
      ④ じぶんのプロジェクトとして保存される
  └ 「できた！」ボタン（ToDo — フェーズ2でクリア記録）
```

各ゲームレッスンに必要なデータ:
- `embed_id` : scratch.mit.edu のプロジェクトID（iframeに使用）
- `project_url` : scratch.mit.edu のプロジェクトURL（新タブで開くリンク）
- `images[]` : 作り方説明のスクリーンショット画像パス
- `description` : 作り方テキスト（ルビ付き）

### 3.3 第1章レッスンページ（概要・操作説明）
- 本文テキスト（**すべての漢字にルビ表示**）
- 説明画像（スクリーンショット等）の埋め込みエリア
- Scratch画面の各部名称説明
- 「つぎへ」「もどる」ボタン

### 3.3 タイピング練習コーナー
- 外部サイト（n-typing.com）への新タブリンクボタン
- スコア手入力欄（フィールド＋「きろく」ボタン）
- スコアは画面に表示（localStorageで一時保存）
- **「つぎのコースをえらんでください」**という案内テキスト表示
- ※ 自分の名前が打てるようになる（ToDo — フェーズ2）

### 3.4 デジタルリテラシー授業（プロローグ・章番号なし）
- アカウントとは何か（説明テキスト＋イラスト）
- Scratchアカウントの作り方（手順ガイド）
- パスワードとは何か
- 良いパスワード・悪いパスワードの例
- パスワードの管理方法（手帳に書く・人に教えてはいけない理由）

### 3.5 UI共通
- 全漢字にHTMLルビタグ（`<ruby>`）でふりがな
- 文字サイズ大きめ（年中〜年長向け）
- かわいいキャラクターをサイドorヘッダーに配置
- パステルカラーのデザイン

## 6. 非機能要件

| 項目 | 内容 |
|------|------|
| 公開環境 | レンタルサーバー（既存）|
| 実装方式 | 単一HTMLファイル（index.html）+ CSS + JS |
| バックエンド | フェーズ1はなし（静的）|
| 対応ブラウザ | Chrome最新版を基準 |
| 操作 | マウス＋キーボード（タッチ不要）|
| 文字 | すべての漢字にルビ（`<ruby>`タグ）|
| デザイン | かわいい系、パステルカラー |
| セキュリティ | フェーズ1は認証なし（誰でも閲覧可）|

---

## 7. 制約・前提

- Scratchのiframe埋め込みは先生がscratch.mit.eduにアップしたプロジェクトを使用
- フェーズ1は認証なし（誰でもURLにアクセスすれば見られる）
- フェーズ1の進捗保存はなし（コンテンツ閲覧のみ）
- タイピング練習は外部サイト連携（スコアのみ手入力で記録）
- 英語学習セクションは作らない
