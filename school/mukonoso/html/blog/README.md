# ブログ記事追加ガイド

このフォルダには、iTeen武庫之荘校のブログ記事が保存されています。

## 📝 新しい記事を追加する方法

### ステップ1: 記事ファイルを作成

1. `ARTICLE_TEMPLATE.html` をコピーします
2. ファイル名を変更します（例: `article-20260215.html`）
   - 日付形式で命名すると管理しやすいです
   - 例: `article-YYYYMMDD.html`

### ステップ2: 記事の内容を編集

`ARTICLE_TEMPLATE.html` を開いて、以下の項目を編集してください:

- **タイトル**: `<title>` タグ内
- **日付**: `.article-date` の部分（YYYY.MM.DD形式）
- **カテゴリ**: `.article-category` の部分
- **記事タイトル**: `.article-title` の部分
- **本文**: `.article-content` 内の内容

### ステップ3: ブログ一覧に追加

#### 3-1. `index.html` を編集

1. `html/blog/index.html` を開く
2. `<div class="blog-list">` の中に、新しい記事のHTMLを追加
3. **記事は上から新しい順に並べる**（最新の記事が一番上）

#### 3-2. 記事のHTML構造（コピーして使用）

```html
<!-- ブログ記事X -->
<article class="blog-item">
    <div class="blog-item-header">
        <span class="blog-date">2026.02.15</span>
        <span class="blog-category">イベント</span>
    </div>
    <h2 class="blog-item-title">記事タイトルをここに入力</h2>
    <div class="blog-item-content">
        記事の要約や冒頭部分をここに入力します。読者の興味を引く内容にしましょう。
    </div>
    <a href="article-20260215.html" class="blog-item-link">続きを読む</a>
</article>
```

#### 3-3. メインのブログページにも追加

`html/blog.html` にも同様に記事を追加してください。

## 🏷️ カテゴリ一覧

以下のカテゴリから選択してください:

- **イベント** - ゲーム大会、クリスマスイベントなど
- **授業レポート** - 授業の様子、生徒の作品紹介など
- **プログラミングのコツ** - 学習方法、検定対策など
- **教室の様子** - 日常の様子、教室の雰囲気など
- **教室長からのメッセージ** - 教室長からのメッセージ
- **お知らせ** - 重要な告知など

## 🎨 特別な強調が必要な記事

重要な記事や特別なメッセージの場合は、背景色を変更できます:

```html
<article class="blog-item" style="background: linear-gradient(135deg, #fff8e1 0%, var(--white) 100%);">
    <!-- 記事内容 -->
</article>
```

## 📷 画像を追加する場合

1. 画像ファイルを `images/` フォルダに保存
2. 記事内で以下のように記述:

```html
<img src="../../images/画像ファイル名.png" alt="画像の説明">
```

## ✅ チェックリスト

記事を追加する際は、以下を確認してください:

- [ ] 記事ファイルを作成した
- [ ] タイトル、日付、カテゴリを正しく設定した
- [ ] ブログ一覧（`index.html`）に追加した
- [ ] メインのブログページ（`blog.html`）に追加した
- [ ] 記事のリンク（href）が正しいファイル名になっている
- [ ] 記事が新しい順（上から）に並んでいる
- [ ] 画像がある場合は、パスが正しい

## 💡 ヒント

- 記事の要約部分は、読者の興味を引く内容にしましょう
- タイトルは具体的で分かりやすいものにしましょう
- 日付は必ず正しく設定してください
- 定期的に記事を追加することで、サイトの更新頻度が上がります
