# 記事投稿エディタ 使い方ガイド

## 📝 概要

マークダウン形式で記事を投稿できるエディタです。管理者認証が必要です。

## 🔐 アクセス方法

1. `/admin/article-editor.html` にアクセス
2. 管理者パスワードを入力（`config.js`の`ADMIN_PASSWORD`で設定）
3. 記事エディタが表示されます

## ✍️ 記事の投稿手順

### 1. 基本情報を入力

- **記事タイトル**: 記事のタイトルを入力（必須）
- **公開日**: 記事の公開日を選択（必須、デフォルトは今日）
- **カテゴリ**: 以下のカテゴリから選択（必須）
  - イベント
  - 授業レポート
  - プログラミングのコツ
  - 教室の様子
  - 教室長からのメッセージ
  - お知らせ

### 2. マークダウンで記事を書く

左側のエディタにマークダウン形式で記事を入力します。

**マークダウン記法の例:**

```markdown
# 見出し1

## 見出し2

### 見出し3

**太字**や*斜体*も使えます。

- リスト項目1
- リスト項目2

1. 番号付きリスト1
2. 番号付きリスト2

[リンクテキスト](https://example.com)

![画像の説明](画像URL)

> 引用文をここに入力できます。

`コード`や

```
コードブロック
複数行
```
```

### 3. プレビューで確認

右側のプレビューパネルで、記事がどのように表示されるか確認できます。

- エディタの内容が変更されると、自動的にプレビューが更新されます
- 「プレビュー更新」ボタンで手動更新も可能
- 「全画面」ボタンで、エディタまたはプレビューを全画面表示できます

### 4. 記事を保存

「記事を保存」ボタンをクリックすると、記事がGoogle Driveに保存されます。

- 保存先: `pentech_info/blog/` フォルダ
- ファイル名: `YYYYMMDD_記事タイトル.html` 形式で自動生成

## 📋 保存後の作業

記事を保存した後、**ブログ一覧に手動で追加**する必要があります。

### ブログ一覧への追加方法

1. `/blog/index.html` を開く
2. `<div class="blog-list">` の中に、新しい記事のHTMLを追加
3. **記事は上から新しい順に並べる**（最新の記事が一番上）

### 追加するHTMLの例

```html
<!-- ブログ記事X -->
<article class="blog-item">
    <div class="blog-item-header">
        <span class="blog-date">2026.02.15</span>
        <span class="blog-category">イベント</span>
    </div>
    <h2 class="blog-item-title">記事タイトルをここに入力</h2>
    <div class="blog-item-content">
        記事の要約や冒頭部分をここに入力します。読者の興味を引く内容にしましょう。
    </div>
    <a href="保存されたファイル名.html" class="blog-item-link">続きを読む</a>
</article>
```

**重要:**
- `blog-date`: 記事の公開日（YYYY.MM.DD形式）
- `blog-category`: 記事のカテゴリ
- `blog-item-title`: 記事のタイトル
- `blog-item-content`: 記事の要約（最初の1-2段落程度）
- `href`: 保存されたHTMLファイル名

## 🎨 特別な強調が必要な記事

重要な記事や特別なメッセージの場合は、背景色を変更できます:

```html
<article class="blog-item" style="background: linear-gradient(135deg, #fff8e1 0%, var(--white) 100%);">
    <!-- 記事内容 -->
</article>
```

## 📁 ファイルの保存場所

記事は以下の場所に保存されます:

- **Google Drive**: `pentech_info/blog/` フォルダ
- **ファイル名**: `YYYYMMDD_記事タイトル.html` 形式

保存後、Google Driveでファイルを確認できます。

## ⚠️ 注意事項

1. **管理者認証が必要**: 記事エディタにアクセスするには、管理者パスワードが必要です
2. **ブログ一覧への追加**: 記事を保存した後、ブログ一覧に手動で追加する必要があります
3. **ファイル名**: ファイル名は自動生成されますが、特殊文字は削除されます
4. **画像のアップロード**: 画像は別途アップロードして、URLを指定する必要があります

## 🔧 トラブルシューティング

### 記事が保存されない

- Google Apps Scriptの権限を確認してください
- Google Driveのアクセス権限を確認してください
- ブラウザのコンソールでエラーを確認してください

### プレビューが表示されない

- マークダウンエディタに何か入力されているか確認してください
- ブラウザのコンソールでエラーを確認してください

### 管理者認証ができない

- `config.js`の`ADMIN_PASSWORD`を確認してください
- セッションストレージをクリアしてから再度試してください
